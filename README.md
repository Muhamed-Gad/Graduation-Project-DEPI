# Healthcare Performance Dashboard | Power BI

## Overview
An interactive Power BI dashboard developed to analyze hospital operational and financial performance using healthcare datasets.

---

## Business Problem
Hospitals often rely on outdated reporting systems that make operational and financial tracking difficult.

This dashboard provides automated and interactive reporting for:
- Executive management
- Operational teams
- Financial analysis
- Patient behavior analysis

---

## Tools & Technologies
- Power BI
- Power Query
- DAX
- Star Schema Modeling
- CSV Datasets

---

## Dataset Information

| Table      | Description          | Records  |
|------------|----------------------|----------|
| encounters | Patient visits       | 27,891   |
| procedures | Medical procedures   | 47,701   |
| patients   | Patient demographics | 974      |
| payers     | Insurance providers  | 10       |
